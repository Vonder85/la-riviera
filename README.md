# STICKY MANAGER FRONT

## Tools

-   React (https://reactjs.org/)
-   Material UI (https://mui.com/)
-   Redux

## Install

```
## INSTALL DEPENDENCIES
yarn
```

## Scripts

### `yarn start`

Start the project in development mode with hot reload enabled.

### `yarn build`

Build the source project to be deployed.

### `yarn test`

Run unit tests.
